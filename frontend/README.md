# HumbleHumansHub: Frontend

This directory contains the Next.js application for HumbleHumansHub.

## ⚙️ Quick Start

1. Install dependencies: `npm install`
2. Configure your environment: `cp .env.example .env.local`
3. Generate the database client: `npx prisma generate`
4. Run the development server: `npm run dev`

For full project documentation, business model, and hackathon details, please refer to the [root README.md](../README.md).
